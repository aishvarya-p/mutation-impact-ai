# AI Mutation Predictor

## Overview

AI Mutation Predictor is a web application that predicts whether a genetic mutation (SNP) is benign or disease-associated using Machine Learning.

The system allows users to input mutation information, sends the data to an AI model through a FastAPI backend, predicts the functional impact of the mutation, and displays the prediction results.

## Features

- Predict whether a mutation is Benign or Pathogenic
- Estimate prediction confidence
- Integrate genomic databases (ClinVar, dbSNP)
- Interactive web interface
- REST API using FastAPI

## Tech Stack

- Python
- FastAPI
- Machine Learning
- Pandas
- NumPy
- Biopython
